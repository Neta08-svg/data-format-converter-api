# Free Data Format Converter API

Free data format conversion API - convert between JSON, CSV, TSV, YAML and XML.

**Live tool & docs:** https://data.wrapper-agency.com

## Quickstart - free API, no key required

```bash
curl "https://data.wrapper-agency.com/api/v1/convert?from=json&to=csv&data=%5B%7B%22a%22%3A1%7D%5D"
```

Clean JSON, open CORS, no signup. Try the interactive tool and full docs at https://data.wrapper-agency.com
